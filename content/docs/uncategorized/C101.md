# 레몬클라우드 스케줄앱



<img src="/images/f6f18b0497a5040217e8d8a373b5ca5d.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />



- 가이드 목차
  - 구글 캘린더 추가
  - 스케줄앱 명령어
  - 슬랙에서 스케줄 추가하기
  - 슬랙에서 스케줄 삭제하기


### 1. 구글 캘린더 추가



- [구글캘린더](https://calendar.google.com)에 접속합니다.
- 좌측 하단의 **다른 캘린더**의 **+** 버튼 -> **캘린더 구독** 클릭
<img src="/images/0a0c055eeb6b45403c4191909647ccc2.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

- developer@lemoncloud.io 를 추가합니다.
<img src="/images/918872e87fa9479d0227a01bf6cfb292.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

- 좌측 하단 다른 캘린더에서 Dev공동개발 확인할 수 있습니다.
- 스케줄앱은 developer@lemoncloud.io 캘린더에 일정을 추가하거나 삭제합니다.


### 2. 스케줄앱 명령어



<img src="/images/96ac05e35d46f2545ea26152fc7a79b0.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

- Description은 생략 가능하며, 이 경우 자동으로 휴가로 등록됩니다.
- Start Date Date는 꼭 yyyy-mm-dd 포맷으로 적어주세요.
- 시차
  - 0.1 = 1시간
  - 0.2 = 2시간
  - 0.3 = 3시간
  - 0.4 = 4시간


### 3. 슬랙에서 스케줄 추가하기



- 위 명령어를 참고하여 슬랙 채널에서 추가할 일정을 입력합니다.
- 아래 예제는 2019년 10월 30일부터 3일동안 휴가 일정을 추가한다는 뜻입니다.
  - 2019-10-30 ~ 2019-11-01
<img src="/images/986ae814fb18f03e420c9fdcca576f36.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

- 명령어 입력 후, 정상적으로 처리될 시 아래처럼 메세지가 출력됩니다.
  - 아래 메세지는 본인만 볼 수 있습니다.
<img src="/images/ed1d176935c8512a8bb152d42fdc0ce7.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />



- #근태 채널에 일정이 등록되었다는 메세지가 발생합니다.
<img src="/images/a27f20186d890f6592ef249b419141b2.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

- 구글 캘린더에 아래 이미지처럼 스케줄이 등록됩니다.
<img src="/images/5ed8b74973a5ca22d44286ef3ab8da73.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />



### 4. 슬랙에서 스케줄 삭제하기



- 일정 등록시 받은 메세지에서 취소 버튼을 클릭합니다.
<img src="/images/1ed0917ea023c6fbe856891bbee579ba.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />



- 정상적으로 처리되면 아래 메세지가 나타납니다.
<img src="/images/0c894b94d894e4d79b094938876bbf63.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />
