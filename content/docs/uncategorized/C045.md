## [구글 개발자 조직 계정 설정(인증)]



구글 개발자 조직 계정으로 가입 후에 계정 설정 까지 완료해야 모든 작업이 완료됩니다.

계정 설정을 위해서는 2가지 작업이 완료되어야 하구요.

매뉴얼을 통해서 인증을 하는 방법을 확인해주세요.

<img src="/images/f61268e83a6588e5b05c6ff78cc4559b.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />



### 1.본인확인



<img src="/images/0d1eeb66013e9a0fdeda943a39d29364.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

*본인 인증에는 조직 증명 서류(사업자등록증) 제출, 본인 인증을 위한 휴대전화 번호 제출, 주소 입력으로 정보를 제출합니다.

**확인 절차 시작하기** 버튼을 누르면 인증에 필요한 정보가 뜹니다. 

**확인 절차 시작하기** 버튼을 누릅니다.



<img src="/images/addab479a9571b713b7d3122f9a3dc24.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

1.**업로드** 버튼을 눌러 조직을 증명할 수 있는 서류를 제출합니다.

*사업자등록증 , 상업 등기 초록, 사업 허가증, 과세 증명서 중에서 제출

2.**다음** 버튼을 누릅니다.



<img src="/images/e78e5853e9e8a78bfbabc59b7dc45f80.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

본인(개인) 신원 정보 입력

3.이름 입력 *대표자로 입력하지 않아도 됩니다. 계정을 등록하는 본인(직원) 개인 정보로 입력해주세요.

4.주민번호 7자리 입력

5.이동 통신사 선택

6.연락처 :핸드폰 번호 입력

7.**다음**을 누릅니다.



<img src="/images/4bf5ea8c04349520271e1d5df796b6f8.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

주소 입력

8.도/시 선택

9.시/군/구 선택

10.상세 주소 입력

11.우편번호 입력

12. **제출**을 누릅니다.



<img src="/images/b7775c60a80914bc0f2cb5a934cd7542.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

13) 인증번호 6자리 코드 입력 후 **확인**을 누릅니다.



<img src="/images/8b75f74702cb9ecb48dea3d7758e961d.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

14)**확인**을** 누릅니다.**

모든 정보 제출이 완료되었습니다. 구글에서 제출된 정보를 검토 한 뒤 승인을 하게 됩니다.



<img src="/images/131e871816684702842973e4a16f1d03.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

보통 1일이내 처리가 완료되는데 승인이 되면 바로 이용 가능하며, 서류가 적합하지 않을 경우 승인을 거절합니다. 

거절되면 받은 메일을 확인해서 조치 사항 확인 후 다시 요청 정보를 제출해야 합니다. 



### 2.조직 웹사이트 인증



<img src="/images/7067d682f2c3d22d43f396957701ec16.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

조직 웹사이트 인증- **세부정보 보기**를 누릅니다.



<img src="/images/6fcaf34a893f5b31fc43e79b91dd0c76.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

1)** 'Search Console로 이동' **버튼을 누릅니다.

등록된 소유자가 Search Console에서 요청을 승인하면 웹사이트가 인증됩니다.

*구글 개발자 조직 계정을 만들 때는 회사 홈페이지(웹사이트)가 있어야 합니다.

웹사이트 인증을 해야 계정 사용이 가능하기 때문에, 홈페이지가 없을 경우 사이트를 만든 뒤 등록 해주세요.



<img src="/images/defe9326086903bd18f13226ab626889.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

2)구글 Search Console에서 **시작하기**를 누릅니다.



<img src="/images/fe1a30e5356c8c852c967d8f24d0a02d.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

**URL 접두어 항목**

3)홈페이지 URL 입력해주세요. http:// 포함하여 입력합니다.

4)**계속**을  누릅니다.



<img src="/images/86a05374d93abc881f288f9558b42758.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

다른 확인방법에 보시면 첫번 줄에 '사이트 홈페이지에 메타태그 추가'방법이 있습니다.

5) HTML태그 선택 - 메타태그 값을 복사해주세요.

복사한 태그값을 사이트 홈페이지(도메인/호스팅)에 추가해시면 됩니다.

사이트에 메타태그 값을 입력을 완료한 뒤 **확인** 버튼을 누릅니다.



<img src="/images/ec7d4c8a2f543a4c1e030e32e59c31dd.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

'소유권이 확인됨' 창이 뜹니다.

6)**완료** 버튼을 누릅니다.



<img src="/images/693fcc1a41598112b8f276c3b5840b21.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />

7)다시 개발자 계정- 내 정보로 이동한 뒤 **확인 요청 보내기**를 누릅니다.

*웹사이트가 확인됨 표시 까지 떠야 완료가 됩니다.



이렇게 2가지 요청사항이 완료되어야, 구글 개발자 조직 계정이 최종적으로 등록이 완료되며 앱 등록을 하실 수 있습니다.



<img src="/images/243c6cd241c8bb837c1016e332cb8b98.jpg" alt="file" width="800" height="400" style="max-width: 100%; height: auto;" />
