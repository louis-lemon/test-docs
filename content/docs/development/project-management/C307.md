## 1. 프로젝트 생성하기(이미지 업로드)



### 1. 레몬에이드에 접속한다. [레몬에이드 바로가기](https://ade.lemoncloud.io/main)

### 2. 레몬에이드 메인화면에서 프로젝트 관리 - 프로젝트 설정 클릭

![레몬에이드에 프로젝트 생성과 라벨링 연동하기 image](https://image.lemoncloud.io/84427ee2-39c1-484e-9882-12d50df0ea29)



### 3. 우측 상단의 + 추가 버튼 클릭

### 4. 새 프로젝트 생성에서 프로젝트 이름(영문)을 입력하고 zip 파일을 업로드

![레몬에이드에 프로젝트 생성과 라벨링 연동하기 image](https://image.lemoncloud.io/58af7748-b128-474d-b7ca-aae998d8c187)



## 2. 카테고리 생성하기



### 1. 프로젝트 관리 - 카테고리 관리에서 우측 상단의 + 추가 버튼 클릭

### 2. 카테고리 이름 입력란에 위에서 입력한 프로젝트 이름과 동일하게 입력 후 색상 선택하고 우측 하단의 생성 버튼 클릭

![레몬에이드에 프로젝트 생성과 라벨링 연동하기 image](https://image.lemoncloud.io/4258360f-3327-487b-a4ca-2fc92be1c971)

## 

## 3. 라벨 생성하기



### 1. 프로젝트 관리 - 라벨 관리에서 우측 상단의 + 추가 버튼 클릭

![레몬에이드에 프로젝트 생성과 라벨링 연동하기 image](https://image.lemoncloud.io/532a24d8-7ee2-4f1c-9ce5-aa52cbb45bdd)



### 2. 라벨링할 라벨 이름(한글 가능)을 입력한 후 카테고리에 위에서 만든 카테고리를 찾아서 선택. 

###  그리고 색상을 선택하고 우측하단의 생성  버튼 클릭

![레몬에이드에 프로젝트 생성과 라벨링 연동하기 image](https://image.lemoncloud.io/353cc385-78c7-4670-ad2f-8c8b5bfa118b)



## 4. 프로젝트 설정하기



### 1. 프로젝트 관리 - 프로젝트 설정에서 설정할 프로젝트 이름 찾아서 클릭 후 하단에 보면 라벨 이름으로 검색에 

###     위에서 만든 라벨  이름을 입력해서 검색하거나 밑에 보이는 라벨 이름들 중에 선택을 한다.

### 그리고 사용 기능에서 사각형에 체크한 후 우측 하단의 수정 버튼을 클릭하면 프로젝트 설정이 모두 끝난다.

![레몬에이드에 프로젝트 생성과 라벨링 연동하기 image](https://image.lemoncloud.io/7683793a-f72d-4810-ba29-ff6a25c54bca)
